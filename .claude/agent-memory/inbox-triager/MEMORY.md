# Inbox Triager - Agent Memory

- `ceo@company.com` — CEO, always Urgent minimum; treat same-day deadlines as hard cutoffs

## Domain Patterns

- `@company.com` — Internal domain; teammates default to Important if they ask a direct question
- `newsletter@techsite.com` — Newsletter sender, default Low

## Recurring Senders

- `newsletter@techsite.com` — "This week in AI" digest, no action required, archive candidate

## User Preferences

- Flag emails where it is unclear if the sender is a direct report vs. peer (affects priority)
- Note time-of-day when evaluating same-day deadlines — if past the stated time, escalate to apology/explanation
