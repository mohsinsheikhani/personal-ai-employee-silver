# Email Templates

## Overview
Reusable email templates with variable substitution for consistent, efficient communication.

## Available Templates

### 1. Cold Outreach (`templates/cold-outreach.md`)
First contact focusing on personalized hooks, credibility proof, value propositions, and low-friction CTAs.

### 2. Follow-Up (`templates/follow-up.md`)
Re-engagement after no response, emphasizing new value additions and easier response options.

### 3. Meeting Request (`templates/meeting-request.md`)
Scheduling calls with clear context, specific agendas, and multiple time options.

## Variable Syntax
Templates use `{{variable_name}}` syntax for consistent substitution patterns.

## Workflow
1. Identify email type from user request
2. Load appropriate template
3. Gather variable values
4. Substitute variables into template
5. Apply tone adjustments
6. Present filled template for review
